Movie API

This is a movie api that can access a list of movie detils.

Method
Request
URL Response
List all movies to user GET /movies List of all available movies
Return available data about specific movie GET /movies[title] A JSON object with movie title
Return a list of movies by genre GET /movies[genres] Returns and a list of movies that match the specified genre
Return Directors Bio GET /movies/[directors name] Gives information about specific director
Sign up a new user POST /users Allows a user to register a new account
User login / /login User can login to their account
View users info GET /users/[username] User can view their account details
Delete account REMOVE /users[username] Allows a user unregister and delete their account
Update a users info PUT /users/[user-name] Update a users inforamtion
Add to favorites POST /users[username]/Movies[MovieID] Adds a movie to the users Favorites list
Remove from Favorites REMOVE /users[username]/Movies/remove[MovieID] Removes a movie from the users favorites list
