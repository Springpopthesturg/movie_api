Movie API

This is a movie api that can access a list of movie detils.

Description endpoints:

Request URL Method Body data Response
Display Welcome Page to the user. / GET None A text message welcoming the user
Register user /users POST A JSON object holding data about the user. A JSON object contain data about user that was register.
Deregister user. /users/[USERNAME] DELETE None A text message indicating user was successfully deregister.
Get the list of favorites movies. /users/[USERNAME]/favorites GET None A JSON object contain data about list of favorites movies.
Add movie to the list of favorites. /users/[USERNAME]/favorites POST A JSON Object holding data about the movies to add to the favorites list. A JSON object contain data about the movie add to the list.
Remove movie to the favorite list. /users/[USERNAME]/favorites/[MOVIE] DELETE None A text message indicating movie was successfully remove.
Get list movies /movies GET None JSON object contain data about all the movies. Structured like:
Get list movies by title. /movies/[TITLE] GET None A JSON object contain data(description, genres, director,image URL) about the movie chose.
Get list directors. /directors GET None JSON object contain all the directors.
Get list director by name. /director/[NAME] GET None A JSON object contain data about director chose, bio, holding name, birthyear and movies.
Get list genres /genres GET None A JSON object contain data about all the genres.
Allow user to login to the application /login POST None A JSON object contain data about the user logged to the app, holding name, lastname, birthday, email, username.
